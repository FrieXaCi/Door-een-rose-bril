Zoals ik al vertelde in mijn vorige blog, er is meer.
Want een HTML-pagina ziet er erg saai uit, hij kan wel wat stijl gebruiken. Gelukkig is hier een oplossing voor genaamd: 'cascading style sheet' of in het kort: 'CSS'. Dit is een laag die je over je HTML legt.

CSS is net als HTML een opmaaktaal en geen programmeertaal. Ik kan hiermee de lay-out stijlen en foto's en animaties toevoegen De mogelijkheden lijken onbeperkt.

Net als bij HTML werk ik met een tekst editor zoals kladblok, ik sla deze op met de extensie .css. Het is gebruikelijk om dit document dan style.css te noemen. Deze style-sheet koppel ik aan mijn HTML pagina en met een code-editor kan ik beide tegelijkertijd bewerken en aanpassen.

Binnen CSS werk ik met element, class en id. Dit zijn de selectors en hiermee selecteer ik de elementen die ik wilt stijlen. Ik kan een element direct stijlen, dit geldt dan meteen voor alle andere dezelfde elementen. Dit is erg handig als je wilt dat elementen op elke pagina van je website dezelfde stijl hebben.

Als ik een element anders dan de rest wilt stijlen, kan ik een class toevoegen aan het element. Als ik meerdere elementen wilt stijlen kan ik deze in een div plaatsen. Een div werkt met een begin- en een eind-tag. Binnen deze div plaats je de elementen die je wilt stijlen. Zelfs binnen een div kan je een div plaatsen. De mogelijkheden zijn dus eindeloos.

Als ik een uniek element heb wat ik wil stijlen kan ik deze een id geven. Een id betekent dat het een uniek element is.

Goed, ik heb een div,
een class en elementen.
Hoe nu verder?

De selectors die ik hebt gemaakt ga ik opmaken in mijn stijlsheet. In mijn style-sheet geopend vanuit mijn code-editor kan ik mijn selectors aanpassen. Als ik een element wil stijlen dan kies ik het element en open en sluit ik deze met gekrulde accolades, deze worden ook wel curly brackets genoemd. Binnen deze accolades schrijf je de stijl. Dit kan variÃ«ren van achtergrondkleur tot de grootte van de pagina die het in mag nemen. Ditzelfde geldt voor de class en het id. Een class begint met een punt: '.' en daarna de naam van de class. Bij id gebruik je een hekje: '#' en dan de naam van het id. Om alle elementen binnen de pagina te stijlen gebruik je een sterretje: '*'.

Op mijn stylesheet ziet het er dan zo uit:
.element {
'Binnen deze curly brackets kan ik de code schrrijven'.
}

Er zijn vele manieren om een pagina te schrijven in HTML en CSS. Het is belangrijk dat ik mijzelf een fijne manier van werken aanleer. Ik moet er wel rekening mee houden dat mijn HTML en CSS leesbaar blijft voor andere developers. Mocht ik dan hulp nodig hebben dan kan ik snel en gemakkelijk hulp krijgen van andere developers.

Het is bijna onmogelijk om alle elementen en attributen uit mijn hoofd te leren, ik zal dus in het begin vaak op zoek zijn naar het juiste element of het juiste gebruik van een class.

Het is wat dat betreft net als een gewone taal vooral veel oefenen en doen. Hoe meer ik oefen hoe meer ik er achter komt dat geheime taal die ik zag in het begin van mijn cursus eigenlijk best leesbaar is.
